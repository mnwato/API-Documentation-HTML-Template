مستندات وب سرویس بورس ترند

به مستندات وب سرویس های بورس ترند خوش آمدید.
پیش از استفاده از API بورس ترند اطمینان حاصل کنید که با قوانین بورس ترند و همچنین قوانین و شرایط استفاده از api بورس ترند آشنایی کامل دارید.
همچنین با توجه به احتمال ایجاد تغییرات در بستر بورس ترند یا ساختار و جزئیات APIها، همواره به اطلاعیه های سایت و کانال رسمی تلگرام بورس ترند دقت کنید. به علاوه به صورت مستمر صفحه سابقه تغییرات API بورس ترند را رصد کندی تا از تمامی تغییرات API با خبر شوید.

برای استفاده از API های زیر نیاز به ایجاد حساب کاربری و خرید تعرفه متناسب با نیاز خود دارید. برای این منظور به صفحه اصلی سایت مراجعه کنید.

احراز هویت

برای احراز هویت در ریکویست های ارسالی باید از توکن صادر شده در حساب کاربری خود استفاده نمایید.
برای این منظور مشابه نمونه کد روبرو در هدر درخواست ها باید اطلاعات زیر را بفرستید.

{"Authorization": "Token 2be7160838c17b25f0cb82cf230f627c45cb2c1a"}

محدودیت ها

جهت ارائه خدمات بهتر و پایدارتر محدودیت هایی وجود دارد که در قسمت توضیحات هر کدام از APIها توضیح داده شده است.

در صورتی که نیاز به سرویسی با شرایط متفاوت دارید لطفا از طریق راه های ارتباطی سایت به پشتیبانی اطلاع دهید.

راهنمای کار با API

اگر برای اولین بار از API نوبیتکس یا سایر بازارهای رمزارز استفاده می‌کنید، برای شروع کار پیشنهاد می‌شود گام‌های زیر را طی کنید:

  • برای شروع کار با API ابتدا در سایت بورس ترند ثبت نام کنید.
  • پس از ثبت نام با ورود به پنل کاربری میتوانید توکن خود را دریافت کنید. دسترسی به این توکن به منزله دسترسی کامل به تمام APIها است. لذا در حفاظت از آن دقت کامل داشته باشید.
  • همچنین توجه داشته باشید که برای استفاده از توکن باید یکی از پلن های بخش تعرفه هارا خریداری نمایید.
    (برای تست سرویس ها میتوانید از پلن رایگان استفاده نمایید)

اندپوینت نماد خاص

متد این اندپوینت GET است

https://bourse-trend-api/api/single 

                API Endpoint

                https://bourse-trend-api.ir/api/single

برای دریافت کلیه اطلاعات مربوط به یک نماد خاص از این درخواست استفاده نمایید:

  • درخواست: GET /api/single/instcode
  • محدودیت فراخوانی: 60 درخواست در دقیقه
  • نیاز به ارسال توکن: دارد
برای دریافت پاسخ حتما از متد GET و با ارسال کد نماد مورد نظر در url استفاده نمایید.
نمونه ای در شکل صحیح این درخواست در روبرو آورده شده است.

    در این قسمت لیستی از خطاهای احتمالی آورده شده است:

    کد خطا توضیح
    00X Some parameters are missing. This error appears when you don't pass every mandatory parameters.
    10X Unknown or unvalid secret_key. This error appears if you use an unknow API key or if your API key expired.
    20X Unvalid secret_key for this domain. This error appears if you use an API key non specified for your domain. Developper or Universal API keys doesn't have domain checker.
    30X Unknown or unvalid user token. This error appears if you use an unknow user token or if the user token expired.
